Installation Instructions (Android Phone)

Download this apk file and install it in your phone. Next start the app and start exploring.

Installation Instructions (Alternative)

If your phone does not support android, then the easiest way to run the app is through an emulator, on your pc. Throughout testing, we used Android studio to simulate a phone, and we recommend it. Follow the instructions here for a guide on how to set up android studio. Once android studio is installed, you can run the app following these instructions. Inside the emulator you can manually download the apk file just as in the previous section, or you can build the app from the source files available in our git repository (If the browser fails with 404 then you lack permission to view the repo, so download the apk file as above)
